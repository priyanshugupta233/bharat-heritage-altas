## 📌 About the Project

**Bharat Heritage Atlas** is a frontend web project designed to make learning about Indian heritage interactive and engaging.

Instead of reading static information, users:
- Click currency notes (₹50, ₹100, ₹200, ₹500)
- Discover historical places behind them
- Explore locations on Google Maps
- Test knowledge through quizzes
- View a leaderboard system

## ⚙️ How It Works

1. User opens the application
2. Selects a currency note
3. App displays a related heritage location
4. User can view location on Google Maps
5. User plays quiz to test knowledge
6. Score is saved in leaderboard
